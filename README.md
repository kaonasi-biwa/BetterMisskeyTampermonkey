# BetterMisskeyTampermonkey
Tampermonkey用のコードです(そうじゃなくてもできるとは思う)  
ReleaseのBetterMisskey.user.jsを使用してください  
## デバッグ済みサーバー
- mi.ablaze.one
- misskey.io
- misskey.04.si
- misskey.sda1.net
- misskey.dev
- submarin.online
## 使用できるサーバー (Tampermonkeyの場合)
- mi.ablaze.one
- misskey.io
- misskey.04.si
- ktnfm.com
- otoya.space
- misskey.sda1.net
- misskey.dev
- submarin.online
- misskey.flowers
- voskey.icalo.net

それ以外の場合も、`// @match        *://<サーバーのドメイン(misskey.ioみたいな)>/*`を追加することで使用することができます  
(報告していただけるとありがたいです)
## 実装機能
- ノートの背景をクリックしたらTwitterのようにノートのページに移動
- フォロー・フォロワーのアイコンをクリックでその人のプロフィールに移動 (submarin.onlineではもとから存在するのでそれ以外)
